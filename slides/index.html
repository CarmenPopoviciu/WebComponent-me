Registering a new custom element

document.registerElement('the-flash', { prototype: Object.create(HTMLElement.prototype) // optional });
  • .registerElement() teaches the browser about the new element
  • The element's tag name must contain a dash (-)
  • Prototype is the place to add custom functionality

Instantiating the element

Extending elements

To extend an element, .registerElement() expects the name and prototype of the element to inherit from

// extend native elements // aka type extension custom elements document.registerElement('mega-button', { prototype: Object.create(HTMLButtonElement.prototype), extends: 'button' });

Extending elements

To extend an element, .registerElement() expects the name and prototype of the element to inherit from

// extend other custom elements var theFlashProto = Object.create(HTMLElement.prototype); document.registerElement('super-flash', { prototype: theFlashProto, extends: 'the-flash' });

Lifecycle callbacks

  • createdCallback (instance was created and its definition was registered)
  • attachedCallback (element was inserted into the document)
  • detachedCallback (element was removed from the document)
  • attributeChangedCallback (element's attribute was added, changed or removed)

Introducing <template>

The template element allows for the declaration of document fragments which are unused by the document when loaded, but are parsed as HTML and are available at runtime for use by the web page

... allows for the declaration of document fragments which are unused by the document when loaded, but are parsed as HTML and are available at runtime for use by the web page.

Important to remember!

  • Declare fragments of markup
  • Parsed as HTML
  • Not rendered
  • Can be cloned
  • Instantiated at runtime

Declaring a template

  • Content is inert
  • Scripts won't run, images won't load, media will not play, etc.
  • Content is considered not to be in the document

Activating the template

A template can be activated by creating a deep copy of its .content

// select template var template = document.querySelector('#mytemplate'); // add your content template.content.querySelector('img').src = 'the-flash.png'; // clone template content and append it to the document document.body.appendChild(document.importNode(template.content, true));
